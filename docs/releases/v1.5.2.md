# epcsaft 1.5.2

`epcsaft` 1.5.2 is the first PyPI publishing release for the package.

## Install

From PyPI:

```powershell
python -m pip install epcsaft
```

With `uv`:

```powershell
uv add epcsaft
```

Current GitHub release/tag install:

```powershell
python -m pip install "epcsaft @ git+https://github.com/tannerpolley/ePC-SAFT.git@v1.5.2"
```

Editable local checkout:

```powershell
git clone https://github.com/tannerpolley/ePC-SAFT.git
cd ePC-SAFT
python -m pip install -e .
```

Source and editable installs require Python `>=3.9`, a C++ compiler, CMake, and Ninja or another working CMake generator. Python 3.13 is the current smoke-test baseline.

## Changes

- Added a GitHub Actions PyPI publishing workflow using PyPI Trusted Publishing/OIDC.
- Added release/tag and manual-dispatch publishing paths.
- Kept wheel and sdist building inside GitHub Actions before publishing to PyPI.
- Updated user docs so PyPI is the primary install path once publishing is live.
