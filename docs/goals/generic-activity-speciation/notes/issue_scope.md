# Issue Scope

Issue: [#89](https://github.com/tannerpolley/ePC-SAFT/issues/89)

Current scope from the issue body:

- harden generic liquid reactive/speciation solving with ideal/apparent and activity-based modes
- support ideal/apparent mode
- support activity-based mode
- support fixed K mode
- support fitted K mode where schema exists
- emit diagnostics for residual blocks and derivative status
- use ePC-SAFT activities/fugacities where requested

Out of scope:

- MEA-specific speciation APIs
- fitting reaction constants by default
