# Generated Plot Gallery

This folder is the canonical home for tracked Matplotlib-generated PNG outputs.

Use:

```powershell
uv run python scripts/paper_validation/tools/build_analysis_galleries.py
```

Open `docs/plots/index.html` to browse all generated plot groups. The gallery is a single-page file explorer: expand folders in the left sidebar, check one or more folders, and the page displays every PNG in those selected subtrees without navigating to separate folder pages.

To rebuild the gallery and serve it as a localhost site:

```powershell
uv run python scripts/paper_validation/tools/serve_plot_gallery.py
```

The server serves `docs/plots/index.html` at `http://127.0.0.1:8765/` by default, or the next available port if that port is already busy.

Folder roles:

- `paper_validation/`: paper-reproduction and diagnostic figures generated by `scripts/paper_validation/**`.
- `fits/`: MIAC, dielectric, osmotic, and presentation fit/validation figures generated by `scripts/fits/**`.

For CSV-backed figures, keep generated numeric payloads beside their PNGs in a `data/` subfolder. The first slice using this pattern is `paper_validation/2025_Figiel/<figure>/data/`, generated by `scripts/paper_validation/2025_Figiel_analysis/generate_figure_data.py`.

Keep source scripts and input CSV/data files in `scripts/` and `data/`; only generated plot artifacts and gallery HTML belong here.
